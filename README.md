# JWST Flask Gallery

# Overview

JWST Flask Gallery is a web application developed to explore and visualize images from the James Webb Space Telescope (JWST).

The project was designed to make astronomical images more accessible through a simple and interactive web interface.

This project was completed as part of the Electrical and Electronics Engineering program at Marmara University.

# Features

- Web-based image gallery
- Navigation between image categories
- Responsive user interface
- Educational astronomy content
- JWST image exploration
- Technologies Used
- Python
- Flask
- HTML
- CSS
- Project Goals
- Improve accessibility of astronomical content
- Present JWST images through a user-friendly interface
- Create an educational platform for space enthusiasts
- Demonstrate web application development using Flask
- 
# System Design

The application was built using Flask as the backend framework and HTML/CSS for the frontend interface.

The platform provides a structured way to browse and view JWST-related content through a simple navigation system.
